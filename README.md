# 💇‍♀️ Sistema de Gestão para Salão de Beleza

Este é um sistema de gerenciamento de atendimentos para salões de beleza, desenvolvido como uma API REST utilizando o ecossistema Spring Boot e Java 21. O projeto foi projetado para automatizar o fluxo de agendamentos, controle de serviços e organização da agenda de profissionais.

🔗 **Link da API em Produção:** https://sistema-de-salao.onrender.com

---

## 🚀 Tecnologias Utilizadas

* **Java 21 & Spring Boot 3** (Base da API)
* **Spring Data JPA & MySQL** (Banco de dados e persistência)
* **Docker** (Empacotamento em containers virtuais)
* **Render** (Hospedagem da aplicação em nuvem)
* **Aiven** (Hospedagem do banco de dados MySQL em nuvem)
